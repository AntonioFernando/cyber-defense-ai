# Sistema Inteligente de Defesa Cibernética com IA e Big Data

## Objetivo

Desenvolver uma solução automatizada para detecção e resposta a ameaças em APIs críticas utilizando Inteligência Artificial e análise de comportamento.

---

## Cenário

Empresa do setor financeiro em migração para nuvem com aumento de ataques fora do horário comercial.

---

## Componentes

### Geração de Logs

Simulação de eventos e acessos.

### Detecção Inteligente

Isolation Forest para detecção de anomalias.

### Resposta Automatizada

Simulação de SOAR realizando contenção.

### Dashboard

Visualização dos incidentes.

---

## Estrutura

```text
data/
scripts/
results/
dashboard/
diagrams/
docs/
```

---

## Execução

```bash
python scripts/generate_logs.py
python scripts/detect_anomalies.py
python scripts/automated_response.py
python scripts/dashboard.py
```

## Resultados

* Logs simulados
* Incidentes detectados
* Resposta automatizada
* Dashboard visual
